## **How to use CodeBanana**

Codebanana is your personal app and website builder powered by AI. You do not need any coding or tech skills. Just describe what you want to make, CodeBanana will finish the left.

<Card title="Agent" icon="robot">
  Agent is the flagship feature of CodeBanana.
</Card>

Agent operates as an **autonomous virtual developer** capable of deeply understanding project context, planning architectural logic, and executing full-stack engineering tasks—transforming natural language requirements directly into production-ready app.

<Card title="Examples of what you can build on CodeBanana">
  - Prototype a new business idea
  - [Fun games ](https://typing-rain-d44fe89d.codebanana.app/)to pass the time
  - A forum for complaining about work
  - Create a personal to-do list
  - Beautifully crafted PPT
  - Set up an event signup page with reminders
  - Build a survey and see answers instantly
</Card>
