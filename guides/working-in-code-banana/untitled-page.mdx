## Project and chat relationship

Each project in CodeBanana has a corresponding group chat that supports three distinct chat modes for different collaboration needs.

## Adding members

You can expand your project team in two ways:

### Add organization members

Click to add existing members from your organization to the project group chat.

### Invite friends

Use the **Invite friends** feature to bring in external collaborators:

- Invited members join the current group chat directly
- They do NOT automatically join your organization
- This is useful for temporary collaborators or external partners

### Joining organizations

Organization membership is always user-initiated:

- Users must actively click **Join organization** from the top-left menu or Contacts directory
- This ensures intentional organization membership rather than automatic enrollment

## Notifications

The notification center displays all system messages and alerts, including:

- System notifications
- Agent permission notifications
- Organization invitations
- Friend requests
- Recharge/payment notifications

### Quick actions

Notifications include convenient shortcuts to help you take immediate action:

- **View in project**: Jump directly to the project where an agent permission or related message was triggered
- **Upgrade**: Quick access to upgrade your subscription or permissions

These shortcuts allow you to quickly navigate to relevant contexts without searching through your projects.

## Chat

The chat feature enables real-time communication with team members (human-to-human only, no agents).

### Creating new chats

Use the **New Chat** button to start conversations:

- **Private chat (1v1)**: Direct messaging with individual team members
- **Group chat**: Create group conversations with multiple participants

### Group notifications

When members join or leave a group chat, all participants receive broadcast notifications to stay informed of group changes.

### Message actions

Each message supports the following actions:

- **Copy**: Copy message content to clipboard
- **Quote**: Reply to a message with a quote reference
- **Forward**: Send the message to another chat or contact

### Search and discovery

Easily find and connect with team members:

- Search organization contacts
- Search chat history
- Initiate new conversations directly from search results

This makes it simple to start conversations with colleagues or continue existing discussions.
